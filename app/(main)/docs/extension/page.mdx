## Setting up the extension

To setup the extension, you have to create and copy an extension key from your profile to the extension. To do it, you must go to the [/auth/profile/extension_keys](extension keys page) and create a key.

Then copy the key using the copy icon, open the extension by clicking the icon in the browser toolbar and paste the key in the input.
