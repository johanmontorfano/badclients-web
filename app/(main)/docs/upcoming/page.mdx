# Upcoming Features

**Here are the features coming to Bad Clients:**

| Name                    | Description                                                                                                                                           |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| Job Database            | Every job analyzed is studied and contributes to a public knowledge database with feedbacks from users. |
| Custom Analysis Fields  | Hunter users will be able to set custom fields in generated analysis.                                                                                 |
| Efficient List Analysis | Enable efficient analysis in jobs lists to flag bad jobs even more quickly.                                                      |

If you have any suggestions for features to add, [submit an issue on GitHub](https://github.com/johanmontorfano/badclients-web/issues)
